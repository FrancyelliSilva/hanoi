# Torre de Hanói

Projeto web interativo desenvolvido para a disciplina de Matemática Discreta.

---

## Como executar

Abra o arquivo `hanoi.html` diretamente no navegador. Não requer instalação.

## Como jogar

- Clique em um pino para selecionar o disco do topo
- Clique em outro pino para mover
- Objetivo: mover todos os discos do pino A para o pino C sem colocar um disco maior sobre um menor

## Tecnologias

HTML · CSS · JavaScript
